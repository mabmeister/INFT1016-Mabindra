PROJECT TITLE: Eynesbury College - First Trimester

PROJECT DESCRIPTION: The Eynesbury College web project is a multi-page student portal designed to showcase first-trimester courses and academic information while providing a personalized user experience. Developed using semantic HTML5, CSS3, and JavaScript, the site features a responsive layout that includes a dynamic course directory, an interactive wildlife gallery, and a custom theme-switching system that toggles between light and dark modes. By utilizing DOM manipulation and localStorage, the project effectively greets users by name and persists their visual preferences across sessions, demonstrating a strong foundation in modern front-end web development and user-centric design.

HOW TO USE: To use this project, save all HTML, CSS, and JavaScript files in a single folder and open index.html in any web browser to launch the portal. Once the page is loaded, you can navigate between the home and course pages using the menu, update the site's appearance using the Toggle Theme button, and personalize the interface by entering your name in the input field to update the header greeting. The interactive wildlife gallery on the home page allows you to dynamically switch images and descriptions by clicking the animal buttons, while the system automatically saves your theme and name preferences to the browser's local storage for your next visit.
